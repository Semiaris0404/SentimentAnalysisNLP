# Linguistic Failure Modes in Transformer-Based Sentiment Analysis

**CS 4650/7650 — Natural Language Processing | Georgia Tech | Spring 2026**  
**Author:** Zhichang Wang (`wangzcgrace2004@163.com`)

---

## Overview

This project investigates how much labeled data each modeling paradigm needs before becoming competitive on sentiment analysis, and whether fine-tuned encoders and prompted generative models fail on different linguistic phenomena.

**Research questions:**
- As labeled data drops from 100% to <1%, how do fine-tuned encoders compare to prompting?
- Do their error modes differ in linguistically interpretable ways?
